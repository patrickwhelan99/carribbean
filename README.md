## Requirements
#### g++
#### make
#### SFML

## Install dependencies (Fedora)
```bash
sudo dnf install g++ make SFML
```

## Build & Run
```bash
git clone https://github.com/patrickwhelan99/carribbean.git
cd carribbean
make && ./game
```
